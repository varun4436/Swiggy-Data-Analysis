Swiggy Data Analysis Using Python

Project Overview:
This project is based on Swiggy food delivery data analysis using Python. The main objective of this project is to explore restaurant 
and food trends, understand customer preferences, and perform exploratory data analysis (EDA) with the help of visualizations.

The analysis includes:
Top food categories analysis
Restaurant ratings analysis
City-wise restaurant distribution
Food price analysis
Popular cuisines trends
Customer preference insights
Data visualization using charts and graphs

This project was created using Google Colab and Python libraries like Pandas, NumPy, Matplotlib, and Seaborn.

Objectives:
Understand Swiggy food delivery trends
Analyze restaurant and food category data
Find popular cuisines and ratings
Visualize trends using charts and graphs
Practice real-world data analysis using Python

Dataset Information:
Dataset Name: Swiggy Dataset
File Format: CSV
Source: Public Swiggy dataset
Total Records: Multiple restaurant and food entries

Dataset contains:
Restaurant Name
Food Category
Cuisine
Ratings
Price
City
Delivery Information
Menu Details

Tools & Technologies Used:
Python
Google Colab
Pandas
NumPy
Matplotlib
Seaborn

Steps Performed:
Imported required libraries
Loaded dataset using Pandas
Checked missing values and dataset information
Cleaned the dataset
Performed exploratory data analysis
Created visualizations using charts and graphs
Extracted useful insights from the data

Visualizations Included:
Pie Chart for Top Food Categories
Bar Graph for Top Restaurants
Ratings Distribution Chart
City-wise Restaurant Analysis
Cuisine Popularity Graph
Price Distribution Analysis

Key Insights:
Fast food and snacks are among the most popular food categories.
Highly rated restaurants receive better customer engagement.
Metro cities contain a larger number of restaurants.
Some cuisines are more popular across multiple cities.
Price and ratings influence customer preferences significantly.

Conclusion:
This project helped in understanding food delivery trends and improved practical knowledge of data analysis and visualization using Python. 
It also provided experience in handling real-world datasets and generating insights from restaurant and customer data.
